# 장바구니
장바구니 미션 저장소


## 🧺 기능 요구사항
- [ ] 회원가입 기능
  - [ ] username
    - [ ] 영어와 숫자로 이루어져야 한다.
    - [ ] 길이는 3자 이상 15자 이하여야 한다.
  - [ ] password
    - [ ] 영어와 숫자로 이루어져야 한다.
    - [ ] 길이는 8자 이상 20자 이하여야 한다.
  - [ ] phoneNumber
    - [ ] ('-'가 없이) 11자리 숫자이어야 한다.
  - [ ] address
- [ ] 로그인
  - [ ] JWT 기반 토큰인증
- [ ] 회원 정보 조회
- [ ] 회원 정보 수정
  - [ ] password, phoneNumber, address 를 수정할 수 있어야 한다.
- [ ] 회원 탈퇴
- API 스펙은 [API 문서](https://www.notion.so/a00bc92443f04c52a852ce16501e981a) 참고
